# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Репозиторий

Это репозиторий документации — хранилище рабочих заметок и материалов об опыте применения AI-агентов (Claude Code) в корпоративной банковской среде. Исходного кода нет.

## Содержимое

- **`ограничения.md`** — практические ограничения при внедрении LLM в корпоративной среде: медленные модели, ограниченный CLI, отсутствие специализированных LLM для аналитики, ограниченные права на выполнение команд.
- **`Описание взаимодействия с агентом.txt`** — кейс взаимодействия с агентом: добавление истории аудита в `RatingDaoImpl.createOrUpdate()`. Агент выполнил ~80% задачи. Описывает рабочий процесс с `/skills brainstorming` и `/init`.
- **`план работ от нейросети.png`** — скриншот сгенерированного AI плана по изменению Java-кода (`RatingB2B.java`, `RatingDaoImpl.java`).

## Контекст

Целевой стек в описываемых задачах — Java/Spring, паттерн DAO, банковская доменная область (рейтинги контрагентов). Документация ведётся в процессе экспериментов с AI-assisted разработкой.
